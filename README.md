#### 一、功能概述
         本实验实现了将符合要求的给定内容输出为QrCode的功能。用户输入的参数若为长度小于50的字符串或文件则输出相应QrCode，否则输出错误提示信息。

         控制台读入的参数长度需小于50才能输出相应的QrCode，能将文本文件中的每一行输出为QrCode并保存在对应的文件中。
